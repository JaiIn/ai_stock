# .env.example

아래 내용을 실제 프로젝트의 `.env.example`로 저장한다. `.env`에는 실제 값을 넣되 Git에 올리지 않는다.

```dotenv
APP_ENV=local
LOG_LEVEL=INFO
DATABASE_URL=sqlite:///data/app.sqlite3

TOSSINVEST_API_BASE_URL=https://openapi.tossinvest.com
TOSSINVEST_CLIENT_ID=replace_me
TOSSINVEST_CLIENT_SECRET=replace_me
TOSSINVEST_ACCOUNT=

ENABLE_LIVE_TRADING=false
DRY_RUN_ONLY=true

LLM_PROVIDER=none
LLM_API_KEY=
LLM_MODEL=
```
