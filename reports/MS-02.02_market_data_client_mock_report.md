# MS-02.02 Market Data Client Mock 보고서

## 수행 범위

- `TossMarketDataClient` request-only 인터페이스 추가
- 문서에 명시된 orderbook, prices, trades, price-limits, candles 요청 정의
- `PriceSnapshot`, `Candle` 최소 모델과 fake `result` 배열 parsing 추가
- 가격과 OHLCV decimal string을 `Decimal`로 변환
- 실패형 `MockTransport` 기반 무전송 테스트 추가

## 안전성 확인

- 실제 Toss API 및 OAuth 호출 없음
- HTTP 전송 메서드 또는 계좌/자산/주문 API 구현 없음
- 모든 market-data 요청은 accountSeq 없이 생성
- Authorization 원문과 raw headers를 예외 및 보고서에 저장하지 않음
- 테스트 값은 mock/fake 값만 사용

## OpenAPI schema 재검증 필요 사항

- 가격의 change, change rate, volume 필드는 현재 로컬 문서에 없어 모델에 임의 추가하지 않음
- orderbook, trades, price-limits 응답 필드는 불명확하여 request interface만 구현
- candles의 `nextBefore`가 포함되는 실제 result wrapper 구조는 추후 재검증 필요
- `adjusted`의 실제 직렬화 형식과 지원 interval 전체 목록은 추후 재검증 필요
