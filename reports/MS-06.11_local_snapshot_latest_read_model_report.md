## Latest read model 설계

- SQLite URI `mode=ro`로 기존 DB만 열고 `PRAGMA query_only = ON` 적용
- schema initialize 및 insert/update/delete/replace/create/drop/alter 미수행
- immutable dataclass로 StockInfo, latest price, latest candle, latest exchange rate, source counts, completeness flags 구성
- 내부 숫자 타입은 `Decimal`로 유지하고 CLI safe JSON에서만 문자열로 변환
- DB 파일이 없으면 생성하지 않고 `database_not_found` safe failure 반환
- 요청 symbol이 없으면 `symbol_not_found` safe failure 반환
- StockInfo만 존재하고 나머지가 누락된 경우 성공한 partial model과 completeness flags 반환

## Latest 선택 정책

- StockInfo: requested symbol `005930`의 단일 upsert 행 조회
- PriceSnapshot: symbol 일치 행을 `timestamp DESC, id DESC`로 정렬해 1개 선택
- Candle: symbol 및 interval `1d` 일치 행을 `timestamp DESC, id DESC`로 정렬해 1개 선택
- ExchangeRate: `USD/KRW` 일치 행을 `date_time DESC, id DESC`로 정렬해 1개 선택
