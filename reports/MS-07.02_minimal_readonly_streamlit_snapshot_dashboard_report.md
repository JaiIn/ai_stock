## 최소 dashboard 설계

- `app/streamlit_app.py`는 page configuration과 safe section rendering만 담당하는 얇은 entrypoint입니다.
- `src/ai_stock/ui/readonly_snapshot_dashboard.py`는 Streamlit에 의존하지 않는 pure presentation helper입니다.
- helper는 `local_snapshot_latest_read_model`만 호출하고 새 SQLite query, schema initialize, migration 또는 write path를 구현하지 않습니다.
- latest read model의 SQLite URI `mode=ro`와 `PRAGMA query_only=ON` 보장을 그대로 재사용합니다.
- Decimal은 latest read model의 safe conversion을 통해 문자열로 표시됩니다.
- DB raw row 또는 raw response 구조를 view DTO에 포함하지 않습니다.
