## Toss API Client Contract Preflight

The new contract module contains frozen dataclasses and deterministic helpers
only. It does not import HTTP clients, read environment variables, read files,
open DB connections, initialize Toss clients, issue OAuth tokens, create
Authorization Bearer headers, request accountSeq, or implement any order,
account, balance, fill, recommendation, ranking, or trade action behavior.
