# MS-06.06 Local Snapshot DB Git Ignore Hardening Report

## 수행 목적

MS-06.05에서 식별한 정확한 `data/` Git 제외 규칙 누락을 보강하고,
기존 local snapshot DB preflight validator를 현재 repository `.gitignore`
규칙으로 다시 검증했습니다.

이번 단계에서는 실제 DB 파일이나 `data/` 디렉터리를 생성하지 않았고,
live smoke와 실제 API/OAuth 호출도 수행하지 않았습니다.

`# Local databases` 섹션에 정확한 `data/` 규칙을 추가했습니다. 기존
`.env`, `.env.local`, `.env.*.local` 제외 규칙과 전역 DB 확장자 규칙은
제거하거나 약화하지 않았습니다.

추가 또는 확인한 필수 pattern:

- `data/`: 이번 단계에서 추가
- `*.sqlite`: 기존 규칙 유지 확인
- `*.sqlite3`: 기존 규칙 유지 확인
- `*.db`: 기존 규칙 유지 확인

기존 `data/local/*.db`, `data/local/*.sqlite`, `data/local/*.sqlite3` 규칙도
유지했습니다. 정확한 `data/` 규칙이 상위 보호 경계를 제공합니다.
