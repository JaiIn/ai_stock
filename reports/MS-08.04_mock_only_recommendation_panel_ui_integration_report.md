## UI Integration Summary

The Streamlit app now renders a `Mock Recommendation Explanation` panel after
the local selection/status area and before the existing data source summary.
The panel:

- builds a `MockRecommendationInput` from the existing
  `ReadonlySnapshotDashboardView` safe fields only
- uses MS-08.02 `build_mock_recommendation_result`
- uses MS-08.03 `build_recommendation_explanation_view_model`
- validates both the mock result and the ViewModel before display
- falls back to a safe warning if validation fails
- displays only the ViewModel safe summary, safe sections, disclaimers, and
  diagnostics

Visible panel copy includes mock-only, observation-only, not investment advice,
no real recommendation, no order execution, no account access, no live API
access, and no credential required.
