# MS-01.02 OAuth Mock 인증 흐름 보고서

## 수행 범위

- OAuth token 요청/응답 모델 추가
- 만료 시각 및 `expires_in` 표현 추가
- 프로세스 로컬 메모리 토큰 저장소 추가
- 인증 정보 없이 동작하는 `MockTokenProvider` 추가
- HTTP 구현이 없는 `LiveTokenProvider`와 live 요청 차단 추가
- 안전한 token/header 출력 마스킹 추가

## 안전성 확인

- 실제 Toss API 또는 OAuth endpoint를 호출하지 않음
- `ALLOW_LIVE_API=false`에 해당하는 기본 설정에서 live token 요청 차단
- live 허용 설정에서도 실제 발급 기능은 미구현 예외로 종료
- access token과 client credential은 `repr`, 예외 및 안전 출력에 원문 미노출
- 토큰은 파일이나 데이터베이스에 저장하지 않고 메모리에만 보관
