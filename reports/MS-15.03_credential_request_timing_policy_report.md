## credential request timing policy 요약

- Current stage is closed for credential request, credential read, credential use,
  env read, file read/write, OAuth, token issuance, live HTTP, live execution,
  accountSeq, order/account/balance/fills, DB, Streamlit, OpenAI/LLM,
  recommendation, ranking, and buy/sell/hold.
- MS-16.00 is recorded as a symbolic candidate where Toss credential request may
  be considered only after explicit user approval and separate gates.
- OpenAI key, accountSeq, and order scope remain blocked even for the MS-16.00
  candidate.

## known warnings

- `python -m unittest discover -s tests`와 `python -m pytest` 종료 후 Windows
  temp cleanup `PermissionError` warning이 재발했다. 테스트 요약과 exit code는 통과다.
- Standalone pytest는 이번 단계에서 exit code 0으로 완료했으며,
  `PytestCacheWarning` 1건이 발생했다.
- `pytest-cache-files-*` permission warning이 `git status --short`에서 재발했다.
  해당 경로는 Git 추적 대상이 아니다.
- `git diff --check`와 `dev_check.py`에서 일부 파일의 LF/CRLF line-ending warning이
  출력되었으나 whitespace error는 없고 exit code 0이다.
