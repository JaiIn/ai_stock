# MS-01.03 인증 요청 컨텍스트 보고서

## 수행 범위

- `MockTokenProvider`와 연결되는 인증 요청 컨텍스트 factory 추가
- 문서 기준 `Authorization` 및 `X-Tossinvest-Account` 헤더 조립
- accountSeq가 없는 비계좌 요청 컨텍스트 지원
- 계좌 헤더 필수 요청의 누락 검증과 안전한 설정 예외 추가
- `TossClientFoundation`에서 인증 컨텍스트 기반 request 객체 생성 지원
- 컨텍스트 및 request 메타데이터 안전 출력 검증

## 안전성 확인

- 실제 Toss API 및 OAuth endpoint 호출 없음
- 실제 HTTP 전송 메서드 또는 endpoint 전용 호출 메서드 추가 없음
- token과 accountSeq는 `repr`, safe dump, 예외 및 보고서에 원문 미노출
- request body는 안전 요청 메타데이터에 포함하지 않음
- 테스트는 mock token, fake accountSeq 및 실패형 `MockTransport`만 사용
