## 수행 목적

고정된 fake snapshot data를 실제 로컬 SQLite 파일에 한 번 저장한 뒤 같은
파일을 다시 열어 repository 조회와 타입 보존을 검증했습니다.

주문·계좌 관련 경로는 사용하지 않았습니다.

## MS-06.05 preflight 테스트 호환성

DB 존재 여부를 preflight contract 판정 입력으로 사용하지 않도록 했습니다.
대신 caller-supplied synthetic observation으로 파일 부재와 존재 상태를 각각
검증합니다.
