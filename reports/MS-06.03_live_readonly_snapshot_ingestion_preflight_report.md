- exchange rate pair: `USD/KRW`
- candle interval: `1d`
- candle count: `1`
- candle adjusted: `true`
- candle before used: `false`
- StockWarnings included: `false`

## Live API Safety Gate preflight 요약

네 allowed business endpoint를 다음 설정으로 metadata-only 평가했습니다.

- `live_api_enabled=true`: 다음 live 후보의 metadata 평가에만 사용
- `real_order_allowed=false`
- `dry_run_only=true`
- `send_requested=false`

모든 allowed metadata는 다음 조건을 충족했습니다.

- method: `GET`
- requires auth: `true`
- requires accountSeq: `false`
- read-only: `true`
- category: `stock_info`, `market_data`, `market_info`

order, mutation, account/assets/balance/fills metadata는 차단되는 것을 테스트했습니다.
Safety Gate 평가는 request, header, token, client 또는 network socket을 만들지
않습니다.
