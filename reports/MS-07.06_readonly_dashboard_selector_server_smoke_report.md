- 서버 기동 전에 전체 offline 검증을 통과한 경우에만 서버를 실행했습니다.
- 서버는 localhost 포트 `8501`에서 정확히 1회 기동했습니다.
- root와 health endpoint만 localhost HTTP로 확인했습니다.
- 인앱 브라우저는 workspace URI metadata 형식 오류로 연결되지 않아 수동 UI 확인을 수행하지 않았습니다.
